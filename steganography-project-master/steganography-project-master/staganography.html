Generate Steganographic images

Tips to operate:
  1. The first canvas is for the image which you want to be seen by everyone!
  2. The second canvas will contain the image which you want to hide in the first image!
  3. upload the foreground image and background image, and click "generate final image" button to get your image!
  4. always use the foreground image of same or greater dimensions then the image to be hidden
